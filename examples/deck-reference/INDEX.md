## Patterns récurrents observés

- **Titre bicolore** : mot fort en `#51bdcb`, reste en `#333333`, UPPERCASE
- **Section banner** présent sur toutes les slides de contenu
- **Logo Ipanova** haut-droit sur toutes les slides de contenu
- **Hexagone teal** comme conteneur d'icône sur tous les icon-blocks
- **Ghost numbers** dans sommaire (gris clair) et numbered-steps (teal 10%)
- **Chevron teal** décoratif bas-droit sur content-split et content-icon-list
- **Fiche référence** = structure systématique Contexte / Intervention / Résultats / Technologies / Key Numbers
